# 1. Инструкция по запуску
### 1) Установка зависимостей сервера
cd server \
npm install

### 2) Установка зависимостей клиента
cd ../client \
npm install

### 3) Настройка Tailwind
cd client \
npm install tailwindcss @tailwindcss/vite \
npm i @tailwindcss/postcss

### 4) Запуск сервера
cd server \
$env:PORT = 8080; npm start 

### 5) Запуск клиента
cd ../client
npm run dev

### 6) Интеграция с LLM
Локальный запуск через Ollama
ollama serve

# 2. Принятые решения
1) Для улучшения визуального восприятия каждая категория товара имеет свой цвет
2) Цена на карточке объявления выделена синим цветом (визуальное выделение привлекает внимание пользователя)
3) В предоставленном бэкенде отсутствовала сортировка по цене. Было принято решение расширить API
4) Возможность открыть любое изображение на весь экран с сохранением навигации между фото
5) Исходный макет формы содержал избыточное пустое пространство, было принято решение немного отойти от предоставленного макета
6) Реализована адаптивность для различных разрешений
